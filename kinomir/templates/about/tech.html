Технологии, использованные в проекте «КиноМир»

🎬 Стек технологий
  • Python 3.9 — основной язык программирования
  • Django 4.2 — веб-фреймворк
  • SQLite — база данных
  • Bootstrap 5 — CSS-фреймворк
  • HTML/CSS — вёрстка
🎞️ Как создавался проект

Проект «КиноМир» создавался пошагово, следуя лучшим практикам Django:

  • Создана структура проекта и настройки окружения
  • Разработаны модели Review и Genre для хранения рецензий и жанров
  • Реализованы view-функции и шаблоны для отображения контента
  • Добавлена пагинация для удобной навигации
  • Создана система регистрации и авторизации пользователей
  • Реализован поиск по фильмам и режиссёрам с подсветкой результатов
  • Добавлены страницы профайлов кинокритиков и детального просмотра рецензий
  • Добавлена модерация — только администратор может одобрять рецензии
  • А что было дальше, вы, скорее всего, не узнаете
